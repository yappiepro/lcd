# CLT-домостроение — Сайт компании по строительству домов из CLT панелей

## Обзор проекта

Это современный одностраничный веб-сайт для компании, специализирующейся на строительстве домов из CLT (Cross-Laminated Timber) панелей. Проект построен на **Nuxt 3** с использованием **Vue 3**, **Tailwind CSS** и продвинутых анимаций на **GSAP**.

### Основные технологии

- **Фреймворк:** Nuxt 3.16+
- **UI-библиотека:** Vue 3.5+
- **Стилизация:** Tailwind CSS + кастомные CSS-анимации
- **Анимации:** GSAP с ScrollTrigger
- **Плавная прокрутка:** Lenis
- **Типизация:** TypeScript
- **Сборка:** Статический сайт (Nitro preset: static)

## Сборка и запуск

### Установка зависимостей

```bash
npm install
```

### Разработка

Запуск локального сервера разработки на `http://localhost:3000`:

```bash
npm run dev
```

### Продакшн

Сборка статического сайта:

```bash
npm run build
```

Предварительный просмотр продакшн-сборки:

```bash
npm run preview
```

Генерация статического сайта:

```bash
npm run generate
```

## Архитектура и особенности реализации

### Анимации

Проект использует богатую библиотеку анимаций, реализованных через GSAP и кастомные CSS-классы:

1. **Scroll Animations** — появление элементов при прокрутке (`.animate-on-scroll`)
2. **Text Reveal** — посимвольное появление заголовков (`.text-reveal-target`, `.text-reveal-line`)
3. **Counter Animation** — анимация счётчиков чисел (`.counter-animate`)
4. **Parallax Effect** — параллакс для элементов (`.parallax-element`)
5. **3D Tilt Effect** — 3D-наклон карточек при наведении (`.tilt-card`)
6. **Custom Cursor** — кастомный курсор с эффектом увеличения
7. **Smooth Scroll** — плавная прокрутка через Lenis
8. **Noise Overlay** — эффект шума на всём сайте
9. **Gradient Orbs** — плавающие градиентные сферы на фоне

### Компоненты анимаций (`composables/useAnimations.js`)

- `useScrollAnimation()` — инициализация анимаций при скролле
- `useTextReveal()` — текстовые раскрытия
- `useCounterAnimation()` — анимация числовых счётчиков
- `useParallax()` — параллакс-эффекты
- `useTiltEffect()` — 3D-наклон элементов

### Конфигурация Nuxt

- **Base URL:** `/lcd/` — сайт развёрнут в поддиректории
- **Пресет Nitro:** `static` — генерация статических файлов
- **Маршруты:** Все маршруты пререндерятся (`/**': { prerender: true }`)
- **Модули:** `@nuxtjs/tailwindcss`

### Дизайн-система

- **Цветовая палитра:** Тёмная тема с оранжевыми акцентами (`#FB923C`, `#F97316`)
- **Шрифты:** Системные шрифты с лёгкими начертаниями
- **Макет:** Фиксированная максимальная ширина 1600px
- **Адаптивность:** Мобильная версия с упрощёнными анимациями

## Соглашения по разработке

### Именование компонентов

- **PascalCase** для имён файлов компонентов (например, `ContactForm.vue`)
- **Кебаб-кейс** для классов CSS и data-атрибутов

### Стилизация

- Предпочтение отдаётся **Tailwind CSS** для утилитарных классов
- Сложные анимации вынесены в отдельные CSS-файлы в `assets/css/`
- Z-индексы используют высокие значения для слоёв (9000+ для оверлеев)

### Производительность

- Анимации инициализируются только на клиенте (`import.meta.client`)
- Кастомный курсор и плавная прокрутка отключаются на мобильных устройствах
- Видео оптимизировано в формате WebM

### Доступность

- Минимальный размер интерактивных элементов: 44×44px (на мобильных)
- Видео имеет атрибут `playsinline` для корректного отображения на iOS
- Семантическая разметка с использованием `<header>`, `<main>`, `<section>`, `<footer>`

## ProjectRef

Директория `ProjectRef/` содержит референсные материалы:

- **PDF-макеты:** Главная, Контакты, Коттеджи Конаково, Портфолио, Технологии
- **Изображения:** PNG и JPG файлы дизайна
- **HTML-референс:** `Главная html.html` — прототип главной страницы

## Примечания

- Сайт использует `ClientOnly` для компонентов, работающих только в браузере (курсор, скролл)
- Все анимации имеют `once: true` для однократного воспроизведения
- Для продакшн-развёртывания требуется настройка базового URL `/lcd/`
